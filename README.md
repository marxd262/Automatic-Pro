# Automatic Pro

Docs-first release hub for the Automatic Pro GaggiMate profile, built with Astro and Starlight.

## Local development

```bash
npm install
npm run dev
```

## Notes

- `Quellen/` is source material only and is ignored by git.
- The version pages (`/v2/` and `/vit3/`) are the primary download entry points.
- Release publishing steps are intentionally kept in a local maintainer note, not in the public docs.
- GitHub Pages deployment is configured in `.github/workflows/deploy.yml`.
